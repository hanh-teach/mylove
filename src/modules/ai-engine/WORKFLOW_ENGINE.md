# AI Creative Workflow Engine

A robust engine designed to orchestrate complex, multi-step creative workflows (Love Notes, Wedding, Marketing, etc.).

## Components
- **WorkflowOrchestrator**: The entry point for triggering workflows.
- **WorkflowPlanner**: Translates user goals into task-based workflows.
- **TaskExecutor**: Executes individual tasks defined in the plan.
- **Registry/Repository**: Centralized management for reusable workflow definitions.
